
## How to Execute `send_webhook.sh`

### On Linux/MacOS
1. Open a terminal.
2. Make the script executable (if not already):
   ```bash
   chmod +x send_webhook.sh
   ```
3. Run the script:
   ```bash
   ./send_webhook.sh
   ```

### On Windows
1. Install a Bash shell (e.g., Git Bash, WSL, or Cygwin).
2. Open the Bash shell.
3. Run the script directly:
   ```bash
   ./send_webhook.sh
   ```
